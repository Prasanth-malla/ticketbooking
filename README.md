# Ticket Booking System

A web-based movie ticket booking application built with HTML, CSS, JavaScript, and PHP.  
This project allows users to view movies, select seats, sign up, log in, and book tickets online.

---

## 🛠️ Features
- User registration and login system
- Browse movies with posters and details
- Select seats and book tickets
- Password recovery via email
- Admin/user management (view registered users)
- Responsive design for desktop and mobile

---

## 📂 Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Other:** GitHub repository management
